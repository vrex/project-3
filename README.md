Project 3 - AWS RDS MySQL

## About

This project demonstrates how to create and manage a cloud-hosted MySQL database using Amazon RDS on AWS.

## Features

- Created a MySQL database on Amazon RDS
- Connected to the database remotely
- Created tables
- Inserted and retrieved data using SQL

## Technologies

- AWS RDS
- MySQL
- SQL

## How to Connect

```bash
mysql -h <RDS-ENDPOINT> -P 3306 -u <USERNAME> -p
```
